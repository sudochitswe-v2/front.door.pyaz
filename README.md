# KyiPoh

- Search movies by title and filter by genre.
- Paginated results with responsive grid layout.
- Dedicated video player page with ArtPlayer integration.
- Theme initialization and optional dev‑tools blocking for production builds.
- Utility hooks for debouncing and theme management.

![Project screenshot](screenshot.png)

## Quick start

```bash
npm install           # install dependencies
npm run dev           # start Vite dev server (http://localhost:5173)
npm run build         # production build
npm test              # run test suite (if any)
```

For more details about the architecture, see the source files under `src/`.
